# Google Books with MERN

#### This website allows the user to query the Google Books database and select from the results to save books to their own bookshelf. 


### Technologies Used:
* React
* Node
* Express
* Javascript/es6
* MongoDB
* Mongoose
* Axios

### Next Steps
